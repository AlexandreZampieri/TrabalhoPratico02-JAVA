# Lista de Exercícios – Estruturas Básicas e Matrizes em Java

## Descrição

Esta lista de exercícios tem como objetivo desenvolver habilidades fundamentais em programação utilizando a linguagem Java, com foco em entrada de dados, validação, estruturas de repetição, vetores e matrizes.

Os exercícios foram elaborados para reforçar conceitos essenciais da lógica de programação, além de introduzir o aluno ao tratamento de dados estruturados, como arrays e matrizes de diferentes dimensões.

## Objetivos

* Praticar entrada e validação de dados via teclado;
* Trabalhar com estruturas condicionais e de repetição;
* Manipular vetores e matrizes;
* Realizar cálculos básicos como soma, média, maior e menor valor;
* Desenvolver raciocínio lógico na resolução de problemas;
* Implementar operações com matrizes, como transposição e inversa;
* Compreender conceitos matemáticos aplicados à programação, como determinante de matrizes.

## Conteúdo Abordado

### Estruturas básicas

Os primeiros exercícios focam na leitura de valores, validação de entrada e uso de estruturas condicionais para garantir que os dados atendam às restrições propostas.

### Vetores e cálculos

São trabalhados conceitos como:

* armazenamento de múltiplos valores;
* cálculo de soma e média;
* identificação de maior e menor valor;
* análise de dados (porcentagem de valores positivos e negativos).

### Matrizes

A segunda parte da lista introduz o uso de matrizes, incluindo:

* criação e exibição de matrizes de diferentes dimensões;
* manipulação de dados em matrizes;
* operações matemáticas com matrizes.

### Operações com matrizes

Os exercícios finais abordam:

* multiplicação por constante;
* transposição de matrizes;
* cálculo de matriz inversa;
* cálculo de determinante.

## Organização dos Exercícios

A lista é composta por 11 exercícios, organizados de forma progressiva:

1. Validação de entrada de dois valores;
2. Leitura e análise de dez valores positivos;
3. Manipulação de N valores com cálculos e estatísticas;
   4 a 6. Introdução a matrizes com valores numéricos e textos;
   7 e 8. Operações com matrizes utilizando constante multiplicativa;
4. Cálculo da matriz transposta;
5. Cálculo da matriz inversa;
6. Cálculo do determinante de uma matriz.

## Considerações Finais

A resolução desta lista contribui para a consolidação dos fundamentos da programação e prepara o aluno para desafios mais avançados, envolvendo estruturas de dados e algoritmos mais complexos.

É recomendável que cada exercício seja implementado de forma clara e organizada, priorizando a legibilidade do código e a correta validação das entradas do usuário.
